PHASE 10 — IFRAME AND EXTENSION TAMPER GUARD
====================================================

Create:
client/src/lib/security/iframeGuard.ts

Detect:
- direct open when iframe-only is required
- invalid parent origin
- missing LMS_LAUNCH_TOKEN
- postMessage origin mismatch
- document.referrer mismatch
- unexpected top/self context
- suspicious iframe environment changes where safely detectable

Do not read cross-origin parent DOM directly.
Use safe checks only:
- window.top !== window.self
- document.referrer
- postMessage handshake
- allowed LMS origins
- signed LMS launch token validation

On invalid:
- do not start playback
- report event
- show Access Restricted overlay

Events:
- DIRECT_OPEN_BLOCKED
- INVALID_PARENT_ORIGIN
- POSTMESSAGE_ORIGIN_MISMATCH
- IFRAME_TAMPER_DETECTED
